# chat-bot
A simple and interactive web-based chatbot that provides quick cooking recipes based on user input. This project demonstrates basic frontend development and chatbot logic using JavaScript.

Features
Chat-based user interface
Keyword-based recipe search
Supports multiple recipes (Tea, Pasta, Fried Rice)
Instant responses with smooth scrolling
Enter key support for quick interaction
Clean and responsive UI

HTML – Structure of the web page
CSS – Styling and layout
JavaScript – Chatbot logic and interactivity
